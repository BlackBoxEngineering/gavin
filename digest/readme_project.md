# Project Overview

## Client
Gavin Woodhouse — business mentor and strategic advisor.

## Problem
Searching "Gavin Woodhouse" currently surfaces negative press and media coverage from 2019 onwards. The existing web presence (gwoo.co.uk) is poor quality with weak SEO. This project creates a professional, authoritative static site to reclaim the narrative.

## Goal
Build and deploy a clean, fast, SEO-optimised static website at **gavinroothouse.com** that:
- Ranks well for "Gavin Woodhouse"
- Presents Gavin's story on his own terms
- Establishes him as a credible business mentor

## Stack
- Static HTML/CSS site (no framework dependency)
- Hosted on AWS (Generation 2 static site pipeline)
- Domain: gavinroothouse.com (registered at Name.com, hosted in Route 53)
- Source: this Git repository
